R에서 제공되는 기본 클래스


vector / list / vector arithmetic / indexing


vector의 활용


x = c(1, 2, 5)

x
[1] 1 2 5
length(x)
[1] 3


  • 벡터의 데이터 타입은 모두 동일

  • 문자가 포함되어 있으면, 모두 문자형으로


c(1, "a", 3)
[1] "1" "a" "3"


  • named-vector

  • 벡터의 원소 각 위치에 이름을 부여할수 있음


x = c(a=1, b=2, c=5)
x
a b c 
1 2 5 
x['a']
a 
1


  • named-vector 에서 이름과 값을 추출하기


names(x)
[1] "a" "b" "c"
as.vector(x)
[1] 1 2 5


list의 활용


  • list 는 다양한 데이터 타입을 포함할 수 있음음


list( 1, "a", 3, c(1,2))

[1] 1

[[2]]
[1] "a"

  • list 도 vector 로서의 성질을 가짐

  • vector 가 가장 기본적인 데이터 클래스
